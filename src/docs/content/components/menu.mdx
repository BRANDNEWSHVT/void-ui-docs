# Menu

A dropdown menu for navigation and actions. Features keyboard navigation and shortcut display.

## Preview

<ComponentPreview
  code={`import { Menu, MenuTrigger, MenuContent, MenuItem, Button } from '@radenadri/void-ui'

export default function MenuDemo() {
  return (
    <Menu>
      <MenuTrigger>
        <Button variant="secondary">
          Options <CaretDown size={14} />

        <MenuItem>New File</MenuItem>

## API Reference

### Menu

Root component that manages menu state.

### MenuTrigger

Wraps the element that opens the menu. Does not add any styling.

### MenuContent

<PropsTable

### MenuSeparator

A visual separator between items.

### MenuGroup / MenuGroupLabel

Group related items with an optional label.

## Accessibility

- Arrow keys navigate between items
- `Enter` or `Space` selects an item
- `Escape` closes the menu
- Focus is trapped within the menu when open
