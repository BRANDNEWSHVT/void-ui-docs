# Label

A simple label component for form controls. For more complex forms, consider using `FieldRoot` with `FieldLabel`.

## Preview

<ComponentPreview
  code={`import { Label } from '@radenadri/void-ui'

export default function LabelDemo() {
  return <Label>Email Address</Label>
}`}
>
  <Label>Email Address</Label>
</ComponentPreview>

## Installation

<CodeBlock
  code={`import { Label } from '@radenadri/void-ui'`}
  language="tsx"
/>

## Usage

### With Input

<ComponentPreview
  code={`<div className="space-y-1.5">
  <Label htmlFor="username">Username</Label>
  <Input id="username" placeholder="Enter username" />
</div>`}
>
  <div className="space-y-1.5 w-full max-w-sm">
    <Label htmlFor="username">Username</Label>
    <Input id="username" placeholder="Enter username" />
  </div>
</ComponentPreview>

### Required Indicator

<ComponentPreview
  code={`<div className="space-y-1.5">
  <Label htmlFor="email" required>Email</Label>
  <Input id="email" type="email" placeholder="you@example.com" />
</div>`}
>
  <div className="space-y-1.5 w-full max-w-sm">
    <Label htmlFor="email" required>Email</Label>
    <Input id="email" type="email" placeholder="you@example.com" />
  </div>
</ComponentPreview>

## When to Use

- **Label** - Simple standalone label for basic forms
- **FieldLabel** - Use with `FieldRoot` for forms with descriptions and validation
