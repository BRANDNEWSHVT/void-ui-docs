# Slider

An input for selecting a value from a range by dragging a thumb.

## Preview

  code={`import { Slider } from '@radenadri/void-ui'

export default function SliderDemo() {
  return <Slider defaultValue={50} />
}`}
>

    <Slider defaultValue={50} />
  </div>

### Controlled

<CodeBlock
  code={`import { useState } from 'react'

function ControlledSlider() {
  const [value, setValue] = useState(50)

  return (
    <div className="space-y-2">
      <SliderLabel>Value: {value}</SliderLabel>
      <Slider value={value} onValueChange={setValue} />
    </div>
  )
}`}
  language="tsx"
/>

### SliderLabel

A label component for the slider.

## Accessibility

- Supports keyboard navigation (arrow keys)
- Focus ring is visible when focused
- `aria-valuemin`, `aria-valuemax`, `aria-valuenow` are set automatically
