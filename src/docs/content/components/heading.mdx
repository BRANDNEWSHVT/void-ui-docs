# Heading

A semantic heading component with automatic HTML element selection based on size. Features tight letter-spacing for a modern look.

## Preview

<ComponentPreview
  code={`import { Heading } from '@radenadri/void-ui'

export default function HeadingDemo() {
  return (
    <Stack spacing="sm">
      <Heading size="2xl">Build something amazing</Heading>
      <Text className="text-(--void-muted)">
        A modern React component library for developers.
      </Text>
    </Stack>
  )
}`}
>
  <Stack spacing="sm">
    <Heading size="2xl">Build something amazing</Heading>
    <Text className="text-(--void-muted)">
      A modern React component library for developers.
    </Text>
  </Stack>
</ComponentPreview>

### Custom HTML Element

By default, the heading level is determined by size. Use `as` to override.

<ComponentPreview
  code={`<Stack spacing="sm">
  <Heading as="h1" size="lg">This is an H1</Heading>
  <Heading as="h2" size="lg">This is an H2</Heading>
  <Heading as="h3" size="lg">This is an H3</Heading>
</Stack>`}
>
  <Stack spacing="sm">
    <Heading as="h1" size="lg">This is an H1</Heading>
    <Heading as="h2" size="lg">This is an H2</Heading>
    <Heading as="h3" size="lg">This is an H3</Heading>
  </Stack>
</ComponentPreview>
