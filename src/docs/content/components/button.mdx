# Button

Interactive button component with multiple variants and sizes. Features glow effects on hover and smooth transitions.

## Accessibility

- Buttons are focusable and can be activated with `Enter` or `Space`
- Disabled buttons are not focusable
- Use descriptive text or `aria-label` for icon-only buttons
- Focus ring is visible for keyboard navigation
