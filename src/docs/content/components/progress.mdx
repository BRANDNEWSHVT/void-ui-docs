# Progress

Displays an indicator showing the completion progress of a task.

## Preview

<ComponentPreview
  code={`import { Progress } from '@radenadri/void-ui'

export default function ProgressDemo() {
  return <Progress value={60} />
}`}
>
  <div className="w-full max-w-md">
    <Progress value={60} />
  </div>
</ComponentPreview>

## Installation

<CodeBlock
  code={`import { Progress, ProgressLabel } from '@radenadri/void-ui'`}
  language="tsx"
/>

### Sizes

### ProgressLabel

A flex container for label and value above the progress bar.

## Accessibility

- Uses `role="progressbar"` for screen readers
- `aria-valuenow`, `aria-valuemin`, `aria-valuemax` are set automatically
