# Input

A styled text input component. Use with Field components for labels and validation.

export default function InputDemo() {

}`}

## Usage

### With Field Components

For labels and validation, use with `FieldRoot`, `FieldLabel`, etc.

<ComponentPreview
  code={`import { Input, FieldRoot, FieldLabel, FieldDescription } from '@radenadri/void-ui'

<FieldRoot>
  <FieldLabel>Email</FieldLabel>
  <Input type="email" placeholder="you@example.com" />
  <FieldDescription>We'll never share your email.</FieldDescription>
</FieldRoot>`}
>
  <FieldRoot className="w-full max-w-sm">
    <FieldLabel>Email</FieldLabel>
    <Input type="email" placeholder="you@example.com" />
    <FieldDescription>We'll never share your email.</FieldDescription>
  </FieldRoot>
</ComponentPreview>

### With Error

<ComponentPreview
  code={`<FieldRoot>
  <FieldLabel>Password</FieldLabel>
  <Input type="password" placeholder="Enter password" />
  <FieldError>Password must be at least 8 characters</FieldError>
</FieldRoot>`}
>
  <FieldRoot className="w-full max-w-sm">
    <FieldLabel>Password</FieldLabel>
    <Input type="password" placeholder="Enter password" />
    <FieldError>Password must be at least 8 characters</FieldError>
  </FieldRoot>
</ComponentPreview>

## Accessibility

- Use with FieldRoot and FieldLabel for proper label association
- Error messages are announced to screen readers via FieldError
- Supports standard keyboard navigation
