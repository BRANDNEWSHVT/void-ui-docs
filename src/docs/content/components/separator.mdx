# Separator

A visual divider to separate content sections.

## Preview

<ComponentPreview
  code={`import { Separator } from '@radenadri/void-ui'

export default function SeparatorDemo() {
  return (
    <div className="space-y-4">
      <p>Content above</p>
      <Separator />
      <p>Content below</p>
    </div>
  )
}`}
>
  <div className="w-full max-w-md space-y-4">
    <Text>Content above</Text>
    <Separator />
    <Text>Content below</Text>
  </div>
</ComponentPreview>

<CodeBlock

## Usage

### Horizontal (Default)

<ComponentPreview
  code={`<Stack spacing="md">
  <Text>Section 1</Text>
  <Separator />
  <Text>Section 2</Text>
  <Separator />
  <Text>Section 3</Text>
</Stack>`}
>
  <Stack spacing="md" className="w-full max-w-md">
    <Text>Section 1</Text>
    <Separator />
    <Text>Section 2</Text>
    <Separator />
    <Text>Section 3</Text>
  </Stack>
</ComponentPreview>

### Vertical

Use `orientation="vertical"` within a flex row container.

<ComponentPreview
  code={`<div className="flex items-center gap-4 h-6">
  <span>Home</span>
  <Separator orientation="vertical" />
  <span>Products</span>
  <Separator orientation="vertical" />
  <span>About</span>
</div>`}
>
  <div className="flex items-center gap-4 h-6 text-(--void-text)">
    <span>Home</span>
    <Separator orientation="vertical" />
    <span>Products</span>
    <Separator orientation="vertical" />
    <span>About</span>
  </div>
</ComponentPreview>

## Accessibility

- Uses `role="separator"` for screen readers
- `aria-orientation` is set based on the orientation prop
