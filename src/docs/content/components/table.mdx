## API Reference

All table components accept `className` for additional styling.

### Table

Root table element. Full width by default.

### TableContainer

Wrapper with border and overflow handling for responsive tables.

### TableHeader

Container for header rows (`thead`).

### TableBody

Container for body rows (`tbody`).

### TableFooter

Container for footer rows (`tfoot`). Has background styling.

### TableRow

A table row (`tr`). Has hover state.

### TableHead

A header cell (`th`). Muted text color.

### TableCell

A body cell (`td`).

### TableCaption

Table caption. Appears at the bottom by default.
