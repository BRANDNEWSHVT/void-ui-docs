# Tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

## Preview

<ComponentPreview
  code={`import { Tooltip, TooltipTrigger, TooltipContent, Button } from '@radenadri/void-ui'

export default function TooltipDemo() {
  return (
    <Tooltip>
      <TooltipTrigger>
        <Button>Hover me</Button>
      </TooltipTrigger>
      <TooltipContent>
        This is a tooltip
      </TooltipContent>
    </Tooltip>
  )
}`}
>

    <TooltipTrigger>

## Installation

<CodeBlock
  code={`import { Tooltip, TooltipTrigger, TooltipContent, TooltipProvider } from '@radenadri/void-ui'`}
  language="tsx"
/>

## Setup

Wrap your app with `TooltipProvider` to enable tooltips:

<CodeBlock
  code={`import { TooltipProvider } from '@radenadri/void-ui'

function App() {
  return (
    <TooltipProvider>
      {/* Your app content */}
    </TooltipProvider>
  )
}`}
  language="tsx"
/>

## Accessibility

- Tooltip appears on hover and focus
- Tooltip is announced to screen readers
- `Escape` key dismisses the tooltip
- Follows WAI-ARIA Tooltip pattern
