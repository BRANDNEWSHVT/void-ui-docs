# Tooltip

A popup that displays information related to an element when hovered or focused.

## Preview

<ComponentPreview
  code={`import { Tooltip, TooltipTrigger, TooltipContent, Button } from '@radenadri/void-ui'

export default function TooltipDemo() {
  return (
    <Tooltip>
      <TooltipTrigger render={<Button>Hover me</Button>} />
      <TooltipContent>This is a tooltip</TooltipContent>
    </Tooltip>
  )
}`}
>
  <Tooltip>
    <TooltipTrigger render={<Button>Hover me</Button>} />
    <TooltipContent>This is a tooltip</TooltipContent>
  </Tooltip>
</ComponentPreview>

## Setup

Wrap your app with `TooltipProvider`:

<CodeBlock
  code={`import { TooltipProvider } from '@radenadri/void-ui'

function App() {
  return (
    <TooltipProvider>
      {/* Your app content */}
    </TooltipProvider>
  )
}`}
  language="tsx"
/>

### TooltipProvider

Wraps your app to enable tooltips with shared delay.

### Tooltip

Root component for a single tooltip.

### TooltipTrigger

The element that triggers the tooltip on hover/focus.

### Composable Components

For advanced customization, use the individual parts:

- **TooltipPortal** - Renders content in a portal
- **TooltipPositioner** - Handles positioning logic
- **TooltipPopup** - The visual popup element
- **TooltipArrow** - The arrow element

## Accessibility

- Appears on hover and focus
- Announced to screen readers
- `Escape` key dismisses
- Follows WAI-ARIA Tooltip pattern
