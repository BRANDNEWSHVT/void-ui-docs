# Stack

A flexbox layout component for arranging children with consistent spacing.

## Preview

<ComponentPreview
  code={`import { Stack, Button } from '@radenadri/void-ui'

export default function StackDemo() {
  return (
    <Stack spacing="md">
      <Button>Button 1</Button>
      <Button>Button 2</Button>
      <Button>Button 3</Button>
    </Stack>
  )
}`}
>
  <Stack spacing="md">
    <Button>Button 1</Button>
    <Button>Button 2</Button>
    <Button>Button 3</Button>
  </Stack>
</ComponentPreview>
