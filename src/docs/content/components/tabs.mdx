# Tabs

A set of layered sections of content that display one panel at a time.

## Styling

TabsTrigger includes the following states:

- **Hover** - Background highlight and text color change
- **Active** - Subtle scale down effect on click
- **Selected** - Surface background with shadow
- **Disabled** - Reduced opacity

## Accessibility

- Follows WAI-ARIA Tabs pattern
- Arrow keys navigate between tabs
- `Home` / `End` keys jump to first/last tab
- Tab panels are associated with triggers via `aria-controls`
