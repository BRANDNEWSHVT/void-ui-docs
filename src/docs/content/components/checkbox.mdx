# Checkbox

A control that allows users to select one or more options. Features smooth transitions and indeterminate state support.

## Preview

  code={`import { Checkbox } from '@radenadri/void-ui'

export default function CheckboxDemo() {
  return <Checkbox label="I agree to the terms and conditions" />
}`}
>
  <Checkbox label="I agree to the terms and conditions" />
</ComponentPreview>

### Indeterminate State

Use `indeterminate` for parent checkboxes that have some children checked.

  code={`<Stack spacing="sm">

### CheckboxGroup

A container for grouping related checkboxes with proper spacing.

## Accessibility

- Checkbox is properly labeled using the `label` prop or associated FieldLabel
- Supports keyboard navigation with `Space` to toggle
- Focus ring is visible when focused
- Disabled and indeterminate states are communicated to assistive technologies
