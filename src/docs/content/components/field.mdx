# Field

A composable wrapper for form fields with label, description, and error handling. Built on Base UI Field for accessibility.

## Preview

<ComponentPreview
  code={`import { FieldRoot, FieldLabel, FieldDescription, Input } from '@radenadri/void-ui'

export default function FieldDemo() {
  return (
    <FieldRoot>
      <FieldLabel>Email</FieldLabel>
      <Input type="email" placeholder="you@example.com" />
      <FieldDescription>We'll never share your email.</FieldDescription>
    </FieldRoot>
  )
}`}
>
  <FieldRoot className="w-full max-w-sm">
    <FieldLabel>Email</FieldLabel>
    <Input type="email" placeholder="you@example.com" />
    <FieldDescription>We'll never share your email.</FieldDescription>
  </FieldRoot>
</ComponentPreview>

## Usage

### Required Field

Use the `required` prop on FieldLabel to show a red asterisk.

<ComponentPreview
  code={`<FieldRoot>
  <FieldLabel required>Username</FieldLabel>
  <Input placeholder="Enter username" />
</FieldRoot>`}
>
  <FieldRoot className="w-full max-w-sm">
    <FieldLabel required>Username</FieldLabel>
    <Input placeholder="Enter username" />
  </FieldRoot>
</ComponentPreview>

### With Error

<ComponentPreview
  code={`<FieldRoot>
  <FieldLabel>Password</FieldLabel>
  <Input type="password" placeholder="Enter password" />
  <FieldError>Password must be at least 8 characters</FieldError>
</FieldRoot>`}
>
  <FieldRoot className="w-full max-w-sm">
    <FieldLabel>Password</FieldLabel>
    <Input type="password" placeholder="Enter password" />
    <FieldError>Password must be at least 8 characters</FieldError>
  </FieldRoot>
</ComponentPreview>

### With Checkbox

Helper text displayed below the input.

Error message displayed when validation fails. Styled in red.
