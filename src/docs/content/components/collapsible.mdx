# Collapsible

An interactive component that expands/collapses content with smooth animations.

## Preview

<ComponentPreview
  code={`import { Collapsible, CollapsibleTrigger, CollapsibleContent } from '@radenadri/void-ui'

export default function CollapsibleDemo() {
  return (
    <Collapsible className="w-full">
      <CollapsibleTrigger>
        What is Void UI?
      </CollapsibleTrigger>
      <CollapsibleContent>
        <p className="pt-2 pb-4 text-sm text-(--void-muted)">
          Void UI is a modern React component library built on Base UI 
          and styled with Tailwind CSS. It provides accessible, 
          customizable components for building modern web applications.
        </p>
      </CollapsibleContent>
    </Collapsible>
  )
}`}

## Usage

### Default Open

<ComponentPreview
  code={`<Collapsible defaultOpen>
  <CollapsibleTrigger>
    Click to collapse
  </CollapsibleTrigger>
  <CollapsibleContent>
    <p className="pt-2 pb-4 text-sm text-(--void-muted)">
      This content is visible by default and can be collapsed.
    </p>
  </CollapsibleContent>
</Collapsible>`}
>
  <Collapsible defaultOpen className="w-full max-w-md">
    <CollapsibleTrigger>
      Click to collapse
    </CollapsibleTrigger>
    <CollapsibleContent>
      <p className="pt-2 pb-4 text-sm text-(--void-muted)">
        This content is visible by default and can be collapsed.
      </p>
    </CollapsibleContent>
  </Collapsible>
</ComponentPreview>

### With Custom Trigger

Use `showIcon={false}` to hide the default caret icon.

<ComponentPreview
  code={`<Collapsible>
  <CollapsibleTrigger showIcon={false}>
    <Button variant="outline-secondary" size="sm">
      Toggle Details
    </Button>
  </CollapsibleTrigger>
  <CollapsibleContent>
    <div className="pt-3 pb-4 text-sm text-(--void-muted)">
      Additional details that can be shown or hidden.
    </div>
  </CollapsibleContent>
</Collapsible>`}
>
  <Collapsible className="w-full max-w-md">
    <CollapsibleTrigger showIcon={false}>
      <Button variant="outline-secondary" size="sm">
        Toggle Details
      </Button>
    </CollapsibleTrigger>
    <CollapsibleContent>
      <div className="pt-3 pb-4 text-sm text-(--void-muted)">
        Additional details that can be shown or hidden.
      </div>
    </CollapsibleContent>
  </Collapsible>
</ComponentPreview>
