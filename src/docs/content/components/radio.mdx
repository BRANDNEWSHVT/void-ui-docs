# Radio

A set of checkable buttons where only one can be checked at a time.

## Preview

<ComponentPreview
  code={`import { RadioGroup, RadioItem } from '@radenadri/void-ui'

export default function RadioDemo() {
  return (
    <RadioGroup defaultValue="comfortable">
      <RadioItem value="compact" label="Compact" />
      <RadioItem value="comfortable" label="Comfortable" />
      <RadioItem value="spacious" label="Spacious" />
    </RadioGroup>
  )
}`}
>
  <RadioGroup defaultValue="comfortable">
    <RadioItem value="compact" label="Compact" />
    <RadioItem value="comfortable" label="Comfortable" />
    <RadioItem value="spacious" label="Spacious" />
  </RadioGroup>
</ComponentPreview>

function ControlledRadio() {

  return (

### Radio

Standalone radio button without label. Use RadioItem for labeled radios.

## Accessibility

- Uses native radio group semantics
- Supports keyboard navigation with arrow keys
- Focus ring is visible when focused
- Disabled state is communicated to assistive technologies
