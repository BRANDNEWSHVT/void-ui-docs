# Chip

A compact element for displaying tags, filters, or selections.

## Preview

<ComponentPreview
  code={`import { Chip } from '@radenadri/void-ui'

export default function ChipDemo() {
  return (
    <Stack direction="row" spacing="sm">
      <Chip>Default</Chip>
      <Chip variant="primary">Primary</Chip>
      <Chip variant="success">Success</Chip>
      <Chip variant="danger">Danger</Chip>
    </Stack>
  )
}`}
>
  <Stack direction="row" spacing="sm">
    <Chip>Default</Chip>
    <Chip variant="primary">Primary</Chip>
    <Chip variant="success">Success</Chip>
    <Chip variant="danger">Danger</Chip>
  </Stack>
</ComponentPreview>

### Removable

Pass `onRemove` to show a remove button.

<ComponentPreview
  code={`<Stack direction="row" spacing="sm">
  <Chip onRemove={() => console.log('removed')}>Removable</Chip>
  <Chip variant="primary" onRemove={() => console.log('removed')}>Primary</Chip>
  <Chip variant="danger" onRemove={() => console.log('removed')}>Danger</Chip>
</Stack>`}
>
  <Stack direction="row" spacing="sm">
    <Chip onRemove={() => {}}>Removable</Chip>
    <Chip variant="primary" onRemove={() => {}}>Primary</Chip>
    <Chip variant="danger" onRemove={() => {}}>Danger</Chip>
  </Stack>
</ComponentPreview>
