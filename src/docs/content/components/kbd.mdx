# Kbd

Displays keyboard keys and shortcuts with a realistic key-cap style.

## Preview

<ComponentPreview
  code={`import { Kbd } from '@radenadri/void-ui'

export default function KbdDemo() {
  return (
    <Text>
      Press <Kbd>⌘</Kbd> <Kbd>K</Kbd> to open command palette
    </Text>
  )
}`}
>
  <Text>
    Press <Kbd>⌘</Kbd> <Kbd>K</Kbd> to open command palette
  </Text>
</ComponentPreview>
