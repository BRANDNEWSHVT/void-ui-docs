# Select

A composable dropdown component for selecting from a list of options.

## Preview

<ComponentPreview
  code={`import { Select, SelectTrigger, SelectContent, SelectItem } from '@radenadri/void-ui'

export default function SelectDemo() {
  return (
    <Select defaultValue="apple">
      <SelectTrigger placeholder="Select a fruit..." />
      <SelectContent>
        <SelectItem value="apple">Apple</SelectItem>
        <SelectItem value="banana">Banana</SelectItem>
        <SelectItem value="orange">Orange</SelectItem>
      </SelectContent>
    </Select>
  )
}`}
>
  <div className="w-full max-w-xs">
    <Select defaultValue="apple">
      <SelectTrigger placeholder="Select a fruit..." />
      <SelectContent>
        <SelectItem value="apple">Apple</SelectItem>
        <SelectItem value="banana">Banana</SelectItem>
        <SelectItem value="orange">Orange</SelectItem>
      </SelectContent>
    </Select>
  </div>
</ComponentPreview>

### Controlled

<CodeBlock
  code={`import { useState } from 'react'

function ControlledSelect() {
  const [value, setValue] = useState('react')

  return (
    <Select value={value} onValueChange={setValue}>
      <SelectTrigger />
      <SelectContent>
        <SelectItem value="react">React</SelectItem>
        <SelectItem value="vue">Vue</SelectItem>
        <SelectItem value="angular">Angular</SelectItem>
      </SelectContent>
    </Select>
  )
}`}

Group related items with an optional label.

### SelectSeparator

A visual separator between items or groups.

## Accessibility

- Supports keyboard navigation with arrow keys
- `Enter` or `Space` selects an item
- `Escape` closes the dropdown
- Selected item is marked with a checkmark
