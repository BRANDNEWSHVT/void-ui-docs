### Controlled Example

<CodeBlock
  code={`import { useState } from 'react'

function ControlledPagination() {
  const [page, setPage] = useState(1)
  const totalPages = 5

  return (
    <Pagination>
      <PaginationList>
        <PaginationPrevious 
          disabled={page === 1}
          onClick={() => setPage(p => p - 1)}
        />
        {Array.from({ length: totalPages }, (_, i) => (
          <PaginationItem 
            key={i + 1}
            active={page === i + 1}
            onClick={() => setPage(i + 1)}
          >
            {i + 1}
          </PaginationItem>
        ))}
        <PaginationNext 
          disabled={page === totalPages}
          onClick={() => setPage(p => p + 1)}
        />
      </PaginationList>
    </Pagination>
  )
}`}
  language="tsx"
/>

## Accessibility

- Uses semantic `<nav>` with `aria-label="Pagination"`
- Current page marked with `aria-current="page"`
- Previous/Next buttons have proper `aria-label`
- Keyboard navigation supported
