  const [checked, setChecked] = useState(false)

  return (

      label="Notifications" 
      checked={checked} 
      onCheckedChange={setChecked} 
    />
  )
}`}

## Accessibility

- Switch has proper `role="switch"` attribute
- `aria-checked` reflects the current state
- Supports keyboard navigation with `Space` to toggle
- Label is properly associated with the switch
