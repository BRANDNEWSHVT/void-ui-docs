# Dropdown

A menu that appears on click with a list of actions. Features smooth animations and keyboard navigation.

## API Reference

### Dropdown

Root component that manages menu state.

### DropdownTrigger

The element that opens the dropdown when clicked.

### DropdownContent

Container for dropdown items with animations.

<PropsTable
  props={[

### DropdownItem

<PropsTable
  props={[

    },

### DropdownSeparator

A visual divider between items.

### DropdownGroup & DropdownGroupLabel

Group related items with an optional label.

## Accessibility

- Full keyboard navigation (Arrow keys, Enter, Escape)
- Focus management and trapping
- ARIA attributes for screen readers
